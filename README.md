# Rap Generator FrontEnd
A web app for DeeCamp 2018 Rap Generator.  
Made with Vue.js.  
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
